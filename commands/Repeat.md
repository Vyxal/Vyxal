# `ř` - Repeat

- Arity: 2
- In: a, b
- Out: a repeated b times

Returns `a` repeated `b` times: think multiplying a list/string by an integer in python.

# Usage
```
`abc`3ř║⟨`abcabcabc`⟩

```
