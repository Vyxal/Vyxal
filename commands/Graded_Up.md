# `⍋` - Graded Up

- Arity: 1
- In: a
- Out: graded_up(a)

Returns a list of integers, which represent the order the elements of `a`. Operates like the same symbol in APL.

# Usage
```
12328193⍋            ║⟨⟨0|5|1|3|2|7|4|6⟩⟩
5 1 3 4 7 9 5 2 4 W ⍋║⟨⟨1|7|2|3|8|0|6|4|5⟩⟩
`jlfkdsjf`⍋          ║⟨⟨4|2|7|0|6|3|1|5⟩⟩
```
