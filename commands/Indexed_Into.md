# `Œ` - Indexed Into

- Arity: 2
- In: a, b
- Out: [a[n] for n in b]

Returns `a` indexed at the indices in `b`

# Usage
```
`ahcekqpvmelspaieyaxwle` ⟨1|9|2|4⟩ Œ        ║⟨⟨`h`|`e`|`c`|`k`⟩⟩
`ahcekqpvmelspaieyaxwle` ⟨10|16|18|17|10⟩ Œ ║⟨⟨`l`|`y`|`x`|`a`|`l`⟩⟩
1 10r 3 6 rŒ                               ║⟨⟨4|5|6⟩⟩
```
