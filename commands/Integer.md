# `I` - Integer

- Arity: 1
- In: a
- Out: int(a)

Takes a single value and casts it to an integer. Floats -> integer. String -> integer (if possible).
Iterable -> base 10 number made of each item as a digit

# Usage
```
12.34I      ║⟨12⟩
`69420`I    ║⟨69420⟩ # Notice the lack of quotes
4 2 0 6 9 WI║⟨42069⟩
```
